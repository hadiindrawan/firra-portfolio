# Firra Citra Ayu Rohimat - Portfolio Website

A modern, responsive portfolio website for Firra Citra Ayu Rohimat, a BNSP-Certified Professional Trainer and L&D Specialist.

## Features

- **Responsive Design**: Optimized for mobile, tablet, and desktop devices
- **Modern UI**: Clean, elegant, and minimalist design with soft blue theme
- **SEO Optimized**: Comprehensive meta tags and semantic HTML structure
- **Interactive Elements**: Smooth scrolling, animations, and hover effects
- **Accessibility**: WCAG compliant with keyboard navigation support
- **Performance**: Optimized loading with lazy loading and efficient code

## Technologies Used

- **HTML5**: Semantic structure with proper SEO meta tags
- **CSS3**: Tailwind CSS framework with custom styling
- **JavaScript**: Vanilla JS for interactivity and animations
- **SVG**: Custom placeholder images and icons
- **Font Awesome**: Professional icon library

## Setup Instructions

1. **Clone or download** the project files to your web server
2. **Replace placeholder images** in the `/assets/` folder with actual photos:
   - `profile-photo.jpg`: Professional headshot (recommended: 320x320px, square)
   - `about-image.jpg`: Training/professional environment photo (recommended: 600x400px)
3. **Update contact information** in the HTML file:
   - Email address
   - Phone number
   - LinkedIn profile
   - Social media links
4. **Customize content** as needed:
   - Services descriptions
   - Experience details
   - Skills and competencies
5. **Test responsive design** on different devices

## Image Recommendations

### Profile Photo
- **Size**: 320x320px (square format)
- **Format**: JPG or PNG
- **Style**: Professional headshot with good lighting
- **Background**: Clean, preferably solid or blurred

### About Image
- **Size**: 600x400px (landscape format)
- **Format**: JPG or PNG
- **Content**: Training environment, workshop, or professional setting
- **Quality**: High resolution, well-lit

## Customization

### Colors
The website uses a soft blue color palette defined in the Tailwind configuration:
- Primary: `#0ea5e9` (soft-blue-500)
- Secondary: `#0369a1` (soft-blue-700)
- Light: `#e0f2fe` (soft-blue-100)

To change colors, update the Tailwind configuration in `index.html`.

### Content
Main content areas to customize:
1. **Hero Section**: Personal introduction and call-to-action
2. **About Section**: Professional background and achievements
3. **Services Section**: Offered services and descriptions
4. **Experience Section**: Professional timeline and skills
5. **Contact Section**: Contact information and form

### Styling
Custom styles can be added to `/styles/style.css` or by modifying Tailwind classes in the HTML.
